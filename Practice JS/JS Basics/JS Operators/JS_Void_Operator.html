Void Operator

Active JS Link

The void operator is most commonly used in managing the javascript: URL (s) because it allows the browser to show the end result of the evaluation of the expression rather than the returned value of the evaluated expression on the client side.

Click here to change the background color

href="javascript:void(document.body.style.backgroundColor='#32CD32');"

Inactive JS Link

In some cases, it is not required by a link to navigate anywhere or do anything. To achieve it, we combine the pseudo URL (javascript:) with void(0) as a value of an href, it tells the browser to return/do nothing when that link is clicked.

Click here it will do nothing

href="javascript:void(0)"
